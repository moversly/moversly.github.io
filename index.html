Prerequisites

Before you begin, make sure you have the following prerequisites installed on your system:

  • Node.js: npm is included with Node.js installation.
  • Git: Required for version control and collaboration.

Setting Up Your Development Environment

  1. Install Node.js: Download and install Node.js from nodejs.org. This will also install npm.

  2. Verify Installation: Open your terminal or command prompt and run the following commands to verify that Node.js and npm are installed correctly:

    node -v
npm -v
